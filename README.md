## Unit Testing with Spock framework

### Requirements

- Java 17

> **Note:** Add JAVA_HOME path environment variable with the value of Java 17 folder

### Run

To run application, use the following command:

   ```gradle
   ./gradlew bootRun
   ```

### Test

To run tests, use the following command:

   ```gradle
   ./gradlew test
   ```
